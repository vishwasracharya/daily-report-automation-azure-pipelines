# Daily Report Automation using Azure Pipelines

This repository demonstrates how **Azure Pipelines** can be used for
scheduled **Python-based automation**, beyond CI/CD for deployments.

## What this project shows
- Running Python scripts in Azure Pipelines
- Scheduled + manual pipeline triggers
- Secure configuration using pipeline variables
- HTML report generation
- Email delivery
- Clean separation of logic and configuration

## Use cases
- Operational reporting
- Monitoring summaries
- Automation tasks
- Scheduled health checks

## Disclaimer
This is a **sample project** using mock data.

## Schedule
Runs daily at **10:00 AM IST** using Azure Pipelines cron scheduling.
